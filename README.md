### 注意事项

>[wiki帮助](https://gitee.com/baihu433/Yunzai-Bot-Shell/blob/master/wiki.md)

>打开白狐脚本的命令
```
bh
```
>获取白狐脚本帮助的命令
```
bh help
```
>修复白狐脚本打不开的命令
```
bash <(curl -sL https://gitee.com/baihu433/Yunzai-Bot-Shell/raw/master/install.sh)
```
>删除白狐脚本的命令
```
rm /usr/local/bin/bh
```
